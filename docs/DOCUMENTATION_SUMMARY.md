## 🚀 使用指南

### 查找文档

#### 方式 1: 从主 README 开始
1. 阅读 [根目录 README.md](../README.md)
2. 找到感兴趣的章节
3. 点击链接跳转到详细文档

#### 方式 2: 从文档索引开始
1. 打开 [docs/README.md](./README.md)
2. 根据角色或问题分类
3. 找到对应的文档

#### 方式 3: 直接搜索
```bash
# 在 docs/ 目录搜索关键词
grep -r "关键词" docs/

# 或使用 IDE 搜索功能
```

### 贡献文档

1. 在 `docs/` 创建新文档
2. 按照 Markdown 规范编写
3. 在 `docs/README.md` 添加索引
4. 在主 `README.md` 添加链接（可选）
